# 数据源管理

## 支持数据源

| 类型         | 数据库       |
| ------------ | ------------ |
| 关系型数据库 | Mysql        |
| 关系型数据库 | Oracle       |
| 关系型数据库 | PostgreSQL   |
| 关系型数据库 | SQL server   |
| MPP 数据库   | Apache Doris |
| MPP 数据库   | Hologres     |
