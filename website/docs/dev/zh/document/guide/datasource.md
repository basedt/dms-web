# 数据源管理

## 支持数据源

| 类型      | 数据库             |
|---------|-----------------|
| RDBMS   | Mysql           |
| RDBMS   | Oracle          |
| RDBMS   | PostgreSQL      |
| RDBMS   | SQL server      |
| RDBMS   | Mariadb         |
| RDBMS   | Polardb Mysql   |
| RDBMS   | Polardb Postgre |
| MPP     | Apache Doris    |
| MPP     | Hologres        |
| MPP     | Gaussdb         |
| MPP     | Greenplum       |
| MPP     | Clickhouse      |
| BIGDATA | Apache Hive     |
