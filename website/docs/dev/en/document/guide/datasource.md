# Datasource

## Supported Datasource

| Type  | Database     |
| ----- | ------------ |
| RDBMS | Mysql        |
| RDBMS | Oracle       |
| RDBMS | PostgreSQL   |
| RDBMS | SQL server   |
| MPP   | Apache Doris |
| MPP   | Hologres     |

## Create Datasource

1. Select Datasource Type
   ![datasource_select.png](./images/datasource_select.png)

2. Add a new datasource, using PostgreSQL as an example
   ![datasource_create.png](./images/datasource_create.png)

3. Click 'Test Connection' and confirm
   ![datasource_success.png](./images/datasource_success.png)
