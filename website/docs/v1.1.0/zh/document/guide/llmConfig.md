# 模型配置

## 支持模型列表

- [通义千问](https://help.aliyun.com/zh/model-studio/get-api-key)
- [OpenAI](https://openai.com/)
- [DeepSeek](https://www.deepseek.com/)
- [Ollama](https://ollama.com/)

## 配置模型

在对应大模型官网申请到 API Key 后，在模型设置中配置即可，菜单路径：系统管理->系统设置->模型设置
